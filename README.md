ereader
=======

Effective news site reader.


Install
=======
    
    
    pip install virtualenv
    virtualenv ereader
    . env/bin/activate

    pip install -r requirements.txt

    python setup.py install


Usage
=====

    ereader http://www.bbc.com/news/world-middle-east-34375875

Tests
=====

    make test-urls
